# 📘 Manuel d'utilisation : GPECTview

**GPECTview** est votre outil de prédilection pour transformer des données brutes (fichiers CSV) en analyses visuelles professionnelles. Ce manuel vous guide à travers les fonctionnalités de l'application.

---

## 1. 📂 Chargement et Préparation des données

### Étapes initiales
1.  **Ouvrez les paramètres** : Cliquez sur l'icône **⚙️** en haut à droite.
2.  **Importez votre CSV** : Glissez-déposez votre fichier. L'application détecte automatiquement les colonnes.
3.  **Choisissez votre Thème** : 8 thèmes sont disponibles (Amethyst, Midnight, Vintage, etc.). Le thème influence non seulement l'interface mais aussi l'esthétique par défaut de vos rendus.

### Sélection des données (Interface principale)
*   **Colonnes** : Cochez les cases des colonnes que vous souhaitez inclure dans votre tableau de bord.
*   **Lignes** : Utilisez le sélecteur pour définir exactement sur quelles données l'analyse doit porter (ex: de la ligne 0 à la ligne 50).
*   **Aperçu dynamique** : Le tableau central se met à jour en temps réel pour vous montrer vos données nettoyées.

---

## 2. 📈 Analyses et Graphiques

L'application propose 5 types de graphiques, chacun adapté à des besoins spécifiques.

### 🔹 Répartition croisée (Mode 3D)
C'est le mode le plus complet. Il permet de croiser trois variables simultanément :
*   **📅 Axe X** : Généralement l'année ou une date.
*   **🎨 Couleur / Empilement** : Une catégorie (ex: Métier, Secteur).
*   **🔢 Valeur à sommer** : Une donnée chiffrée (ex: Nombre d'embauches).
*   *Idéal pour : Voir l'évolution des recrutements par métier au fil des années.*

### 🔹 Barres & Barres Empilées
*   **Barres simples** : Pour comparer des fréquences (ex: Nombre de salariés par département).
*   **Empilement** : Pour comparer des sous-groupes au sein d'une même barre.

### 🔹 Barres centrées
Idéal pour les pyramides des âges ou les balances de satisfaction.

### 🔹 Camembert
Parfait pour illustrer des parts de marché ou des répartitions budgétaires simples.

---

## 3. 🔧 Personnalisation du Graphique

Dans le menu **⚙️ Paramètres graphique**, vous pouvez ajuster :
*   **Affichage en %** : Transforme les sommes brutes en parts relatives.
*   **Titres et Axes** : Personnalisez les textes pour qu'ils soient prêts pour vos présentations.
*   **Palette de couleurs** : Changez le jeu de couleurs utilisé dans le graphique indépendamment du thème général.
*   **Inversion d'axes** : Pratique pour les barres horizontales quand les libellés sont longs.
*   **Masquage des zéros** : Nettoie le graphique en supprimant les catégories vides.

---

## 4. 💾 Gestion des Modèles (Configurations)

La barre latérale gauche regroupe vos **Configurations**. Une configuration enregistre tout : colonnes choisies, type de graphe, couleurs, titres, etc.

*   **Sauvegarder** : Une fois satisfait, donnez un nom à votre configuration pour la retrouver plus tard.
*   **Organiser** : Créez des dossiers pour trier vos analyses par projet ou par année.
*   **Adapter** : Si vous ouvrez un nouveau fichier CSV avec des colonnes légèrement différentes, l'application vous proposera d'adapter manuellement vos anciens modèles.

---

## 5. 📷 Exportation des résultats

*   **Export Image (PNG)** : Télécharge le graphique en haute résolution.
*   **Export CSV** : Télécharge le tableau de données correspondant exactement à votre sélection actuelle.

---

> 💡 **Astuce** : Les résultats sont optimisés pour être copiés-collés directement dans des présentations PowerPoint ou des rapports PDF.
